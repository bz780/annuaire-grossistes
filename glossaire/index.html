Glossaire B2B

Définitions des termes du commerce B2B : Incoterms, MOQ, NAF, EAN, Kbis, palette, EUR1. Pour les acheteurs débutants en sourcing professionnel.

MOQ (Minimum Order Quantity)
Quantité minimum d'achat fixée par le fournisseur. Peut s'exprimer en pièces, cartons ou palettes. En B2B France, le MOQ est généralement plus accessible que dans l'import asiatique direct.
Incoterms 2020
Règles internationales définissant les responsabilités acheteur/vendeur dans le transport. Les principaux : EXW (départ usine), FOB (port d'embarquement), CIF (assurance-transport jusqu'au port), DAP (livraison destination), DDP (tout inclus, droits de douane payés).
Code NAF / APE
Code à 5 caractères attribué par l'INSEE classifiant l'activité principale de l'entreprise. Les grossistes ont un code en 46.xx (commerce de gros). Différent du code NAF d'un détaillant (47.xx) ou d'un producteur (selon le secteur).
Kbis
Document officiel délivré par le greffe du tribunal de commerce attestant l'existence légale d'une entreprise commerciale française. Comporte SIREN, dénomination, dirigeants, capital social. Validité 3 mois généralement exigée pour ouverture de compte fournisseur.
EAN / GTIN-13
Code-barres standard à 13 chiffres identifiant un produit de manière unique. Indispensable pour la vente en grande distribution, marketplace (Amazon, Cdiscount), ou inventaire ERP. Les EAN s'achètent auprès de GS1 France.
Palette EUR (palette Europe)
Palette en bois normalisée 1200×800 mm, échangeable dans le réseau européen. Capacité standard : 1000 kg statique, 1500 kg dynamique. Hauteur palettisée maximale en transport routier : 2,40 m.
EUR.1
Certificat de circulation des marchandises attestant l'origine UE pour bénéficier d'accords préférentiels (réduction droits de douane) avec certains pays tiers (Suisse, Norvège, Maghreb, etc.).
DDP / DDU
DDP (Delivered Duty Paid) : le vendeur paie tous les frais jusqu'à la livraison, droits de douane inclus. DDU (Delivered Duty Unpaid) : l'acheteur règle les droits à l'arrivée. À clarifier en amont sur tout achat international.
B2B / B2C
B2B (business-to-business) : transactions entre professionnels. B2C (business-to-consumer) : vente à des particuliers. Régimes fiscaux et obligations (TVA, garanties, étiquetage) différents.
Conditionnement / SU-CU-PU
SU (Sales Unit) : unité de vente individuelle. CU (Carton Unit) : nombre de SU par carton. PU (Pallet Unit) : nombre de CU par palette. À préciser systématiquement dans toute commande pour éviter les malentendus.
